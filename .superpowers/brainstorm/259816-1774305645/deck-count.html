Deck count — how do you want to control it?

Ribs use a count spinner in the Inspector — decks could work the same way, or differently.

A

Deck Count spinner (Inspector)

Same as Rib Count — set a number, decks auto-space evenly between keel and gunwale. Drag handles to fine-tune individual heights.

Rib Count5
Deck Count2
Deck Heights[2.0, 3.5] auto

Handles in viewport let you drag each deck to exact height after.

B

+ / − buttons on the gizmo

A "+" handle at the top of the hull and a "−" on each existing deck let you add or remove a deck directly in the viewport.

A is simpler and consistent with how ribs work. B is more direct but adds more gizmo complexity. You can always do A first.
