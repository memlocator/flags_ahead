Hull Editor — What should be draggable?

Each handle type gives you different control over the ship shape. Which do you want?

A

Hull profile handles

Drag the control points of the cross-section curve — bilge, tumblehome, gunwale. All ribs share one profile shape that scales to fit each rib's height/width.

B

Rib position handles

Drag ribs fore/aft along the ship's length. Also move bow and stern endpoints to lengthen or shorten the hull.

C

Deck height handles

Drag the deck level markers up/down to set where each deck sits inside the hull.
